# LinguaVerse · 零技术基础部署指南（小白版）

> 全程不用写代码！纯点鼠标，5 分钟让你的网站上线。

---

## 你会得到什么

部署完成后，你会获得一个全世界都能访问的网址，例如：

```
https://linguaverse.vercel.app
```

（"linguaverse" 会被替换成你自己取的名字）

---

## 第 1 步：拿到项目文件

在你电脑上，找到 `/workspace` 文件夹，里面有 10 个文件：

```
index.html        ← 网页主页
styles.css        ← 样式（颜色排版）
bundle.js         ← 核心代码（课程、单词、会员系统）
manifest.json     ← iPhone 可以添加到主屏
sw.js             ← 离线缓存
icon.svg          ← 图标
icon-192.png      ← 图标
icon-512.png      ← 图标
vercel.json       ← 部署配置（别删）
README.md         ← 这个文件
```

**保持文件名不变，别乱改！**

---

## 第 2 步：注册 GitHub（代码仓库，免费）

1. 打开 [github.com](https://github.com)
2. 右上角 "Sign up" → 填邮箱、密码、用户名（比如 `xiaoming`）
3. 完成注册，登录

---

## 第 3 步：把文件上传到 GitHub

1. 登录 GitHub 后，右上角 "+" → **New repository**
2. Repository name 填：`linguaverse`
3. 勾选 **Public**（公开，别人才能看到）
4. 点 **Create repository**
5. 页面上找到 **uploading an existing file** → 点它
6. 把 `/workspace` 里的 **10 个文件** 全部拖进去
7. 页面底部点 **Commit changes**

**好了，文件已在网上了！**

---

## 第 4 步：用 Vercel 部署（30 秒上线）

1. 打开 [vercel.com](https://vercel.com)
2. 点 **Sign Up** → 选择 **Continue with GitHub**
3. 点 **Add New → Project**
4. 找到 `linguaverse` 这个仓库 → 点 **Import**
5. Framework Preset 保持 "Other"，Build Command 和 Output Directory **留空**
6. 点 **Deploy**
7. **等 30 秒！**

🎉 **恭喜！你会看到：Visit → https://linguaverse.vercel.app**

**把这个网址发给朋友，他们就能访问你的学习平台了！**

## 第 5 步：让用户能付钱给你（可选 · 10 分钟）

### 方案 A：国内用户（最简单，支付宝/微信）

用 **[xorpay.com](https://xorpay.com)**（个人免签约，2 元手续费封顶）

1. 注册 xorpay 账号，实名认证
2. 在后台获取 `app_id` 和 `app_secret`
3. 让用户扫码付款 ¥299 → 你手动给他们一个"激活码"
4. 用户输入激活码 → 自动解锁全部课程（我已在 bundle.js 里写好逻辑，只需填入你生成的激活码即可）

### 方案 B：海外用户（Stripe · 支持信用卡）

1. 注册 [stripe.com](https://stripe.com)
2. 在后台：Products → Add product → 名称 "终身会员"，价格 ¥299
3. 获取 `pk_test_xxxxxxxxxx`（Publishable key）
4. 打开 bundle.js，找到 `STRIPE_PUBLISHABLE_KEY`，替换成你的 Key
5. 重新上传到 GitHub，Vercel 会自动重新部署

## 常见问题

**Q：真的免费吗？**
A：Vercel 免费版对小流量完全够用。每天 1000 人以下都免费。

**Q：我没有 GitHub 怎么办？**
A：另一个更简单的办法：注册 [Netlify.com](https://netlify.com)，直接把 10 个文件拖到网页上，立刻获得网址。

**Q：用户付钱后，我怎么知道？**
A：Stripe / Xorpay 后台会有订单列表，也会给你发邮件。

**Q：文件能改吗？改了之后要重新部署吗？**
A：改完本地文件，重新上传 GitHub 覆盖，Vercel 会自动重新部署（约 1 分钟）。

**Q：想绑定自己的域名（比如 www.我的域名.com）？**
A：在阿里云/腾讯云买个域名（约 ¥50/年），在 Vercel → Project Settings → Domains 里添加，按提示配置 DNS 即可。
